シミュレーションの開始方法と基本操作
1. Newボタンを押して、シミュレーションを初期化する
2. 各種オブジェクトの設置
3. ▶||ボタンでシミュレーションのスタート・ストップする
4. File > Closeでシミュレーションを終了する


オブジェクトの作成
(注意 シミュレーションウィンドウとコントロールウィンドウが独立しているため、オブジェクトの設置するときには、一度シミュレーションウィンドウをクリックしてアクティブにする必要があります。)

・出口を知っている避難エージェントを設置する(goal agent)
1.speedを入力し、mouse setボタンを押す
2.シミュレーション空間の任意の場所をクリックして設置する

・出口を知らない避難エージェントを設置する(non goal agent)
1.goal id(脱出出口と一致する)を入力しmouse setボタンを押す
2.シミュレーション空間の任意の場所をクリックして設置する(idが正しく入力されていない場合は設置できない)

・複数人を一度に配置する(random spawn agent)
1.エージェントの設置範囲 random rangeを入力
2.agent typeを設定
-goal× 出口を知らないエージェント
-goal● 出口を知っているエージェント
3.agent typeがgoal●の場合は、goal idを入力
4.シミュレーション空間の任意の場所をクリックして設置する

・脱出出口を設置する(Goal)
1.idに任意の文字と、脱出口の大きさとなるwidth,heightの数値を入力し、
2.mouse setをボタンを押す
3.シミュレーション空間の任意の場所をクリックして設置する

・壁を設置する(Obstacle)
1.lineのmouse setボタンを押す
2.シミュレーション空間の任意の場所を一回クリックで壁の始点を設定し、もう一度クリックすることで壁の終点を決定する

オブジェクトの削除
・消しゴム機能(eraser)
1.eraser mouse setボタンを押す
2.消したいオブジェクトの上でクリック(またはクリックしながら)することで削除できる

・すべての設置したエージェントを削除する
delete_agentのdeleteボタンを押す

・すべての脱出出口を削除する
delete_goalのdeleteボタンを押す

・すべての壁を削除する
delete_lineのdeleteボタンを押す

パラメータの各種設定
1.パラメータの入力を変更し、setボタンを押す
(注意 setが押されると、シミュレーションが初期化され、オブジェクトが消えてしまいます。)

・Environment
- background size_x 背景画像の横の長さ
- background size_y 背景画像の縦の長さ
- scale_x シミュレーション空間の横の長さ
- scale_y シミュレーション空間の縦の長さ
- cell_interval 障害物セルの間隔の長さ

・Agent
- radius 体の半径
- speed 1stepあたりの最高歩行速度
- view_length 視野の最大距離
- view_angle 視野の角度

・Potential
- goal_weight 出口を知っているエージェントの出口方向に働く引力の強さ
- goal_range 出口を知っているエージェントの出口方向に働く引力大きさ
- agent_weight エージェント間に働く反発力の強さ
- agent_range エージェント間に働く反発力の大きさ
- obstacle_weight 障害物に働く反発力の強さ
- obstacle_range 障害物に働く反発力の大きさ


その他の機能

・シミュレーションのログを保存する
1. ●ボタンを押して、保存先と保存するフォルダ名を決める
2. シミュレーションをスタートする
3. 任意のタイミングでストップさせ、●ボタンを押すと保存される

保存される情報
エージェントの行動ログ、壁設定ファイル、出口設定ファイル、背景画像、パラメータの設定ファイル

・シミュレーションをログを再生する
1. Fileタブ > Open Simulation Logで保存したシミュレーションログを読み込む
2. ▶ボタンで再生する

・シミュレーションの情報を保存する
1. Fileタブ > Save Work Space で保存先とフォルダ名を決めて保存

保存される情報
エージェント、壁、出口、背景画像、パラメータの設定ファイル


・シミュレーション情報を読み込む
1. Fileタブ > Open Work Spaceで保存していたものを読み込む

・背景画像を読み込む
Fileタブ > Open Background Pictureで任意の画像読み込む(PNGのみ)
※背景画像はパラメータのbackground size_x,yによって縮小、拡大されます


トピックデータの作成と表示方法

ターミナルでの起動を推奨
 Python環境(Python3.6以上)が必要となります。

Python環境のセットアップ方法
requirements.txtをダウンロードして、以下コマンドよりセットアップをしてください。
pip install -r requirements.txt
venvなどの仮想環境を使いたい場合は、仮想環境をアクティブにしている状態でアプリを起動してください。

・トピックデータの作成(トピック3~50までのデータを作成)
1. Topicタブ > Create Corpus Dataを選択
2. ２つのパラメータを入力し、Createボタンをクリック
- map_split_size 集団とみなす範囲(緑の線で表示される)
- step_split_size データを作成する際のステップ間隔
3. シミュレーションログファイルのあるフォルダを選択すると、そのフォルダに必要なデータが作成される(データの作成が完了すると自動でCreate Corpus Settingsのウインドウが閉じます)

・トピックの表示
1. Topicタブ > View Topic より作成したトピックデータフォルダを選択する(topic_kとなるフォルダ名)
2. Select Topicウインドウでトピックの表示設定を変えることができる
